SACHIDAX Packet Tracer Learning Map

A structured, interactive learning system designed to help beginners understand networking fundamentals and build hands-on skills in Cisco Packet Tracer. This project combines an interactive learning tree, detailed explanations, command references, and a full 30-day guided roadmap for progressive skill development.

Features
Interactive Learning Tree

All networking topics are arranged in a collapsible tree format for fast and clear navigation. Each node includes:

Hinglish explanations for easier understanding

Full forms of networking abbreviations

Practical examples

Key commands and usage notes

Common troubleshooting insights

30-Day Mastery Roadmap

A structured, beginner-friendly study plan with daily goals. Each day includes:

Focus topic

Practical tasks

Lab instructions

Command practice

Progress tracking using browser LocalStorage

Command Reference

A complete command guide covering:

Switch basics

IP addressing and interface configuration

VLAN and trunking

Inter-VLAN routing

OSPF and EIGRP

DHCP, DNS, and NTP

NAT and ACL configuration

SSH and basic device hardening

Troubleshooting commands

Commands are explained with purpose, usage, examples, and expected results.

Clean UI and UX

Optimized for readability and learning:

Lightweight layout

Hover-based explanations

Expandable sections

Smooth transitions

Mobile-friendly design

No external libraries required

Purpose

This project aims to provide an accessible and practical learning resource for:

Beginners studying networking

Students preparing for CCNA

Entry-level SOC L1 and NOC roles

IT support staff improving core networking knowledge

Self-learners seeking a structured learning path

The goal of the SACHIDAX project is to make high-quality networking education simple, clear, and independent of paid training.

Tech Stack

HTML5

CSS3

Vanilla JavaScript

LocalStorage for progress tracking

No frameworks or external dependencies are required.

How to Use

Clone or download the repository.

Open index.html in any modern web browser.

Explore topics using the learning tree.

Follow the 30-day study plan to build practical skills.

Use the commands section during labs and practice sessions.

Reset progress at any time from the UI.
